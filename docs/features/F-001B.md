# F-001B: Platform Dependency Inventory

## Objective

Inventory legacy platform dependencies that block modern Apple silicon compatibility.

## User Value

This identifies Carbon, QuickTime, classic Mac resource APIs, old Mac types, and build settings that must be replaced or isolated.

## Non-Goals

- Do not replace platform APIs.
- Do not update the Xcode project.
- Do not convert assets.

## Acceptance Notes

- Platform dependencies are grouped by concern: UI/events, rendering, audio, resources, files, build settings, and language/runtime assumptions.
- Each dependency group includes representative files or project locations.
- High-risk unsupported dependencies are clearly marked.

## Affected Systems

- Legacy source files: `Sources/`
- Modernization harness: none expected
- Resources/assets: inventory only
- Build or tooling: `Ultima3.xcodeproj/project.pbxproj`
- Documentation: `docs/modernization/legacy-inventory.md`

## Validation Plan

Use source searches for imports, framework names, classic Mac types, and project settings. Record commands or search terms used.

## Open Questions

- Which dependencies can be isolated behind adapters first?
- Which dependencies are only build-time assumptions versus runtime behavior?
